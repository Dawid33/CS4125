# Library Management System

Run dev app

```
flask --app dev.py --debug run -p 8080
```

Run prod app

```
python app.py
```

Manual command line setup

Linux
```bash
python -m venv ./venv
source ./venv/bin/activate
python -m pip install -r requirements.txt
```

Windows
```powershell
python -m venv ./venv
venv\Scripts\activate
pip install -r requirements.txt
```
